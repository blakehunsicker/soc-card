Things to remember

When the user uploads a background image:

  1. Apply text-shadow:0px 0px 12px #AAA; to all text except the caption.

  2. Enable the image overlay dropdown, which will (unless you know a better way) control the opacity of a #000 overlay on top of the image.

    The options are:
    Dark: background:rgba(0,0,0,0.5)
    Medium: background:rgba(0,0,0,0.25)
    Light: background:rgba(0,0,0,0.1)
    None: background:rgba(0,0,0,0.0)


If Results Title is longer than 40 characters, make it H2:

Disable font size dropdown and, if necessary, change from H1 to H2.
